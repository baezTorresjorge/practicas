Linux

Linux es un sistema operativo y un nucleo.
Es un de los paradigmas del desarrollo de software libre (y de codigo abierto) donde el codigo fuente
esta disponible publicamente y cualquier persona puede libremente usarlo, modificarlo y/o distribuirlo.
El sistema operativo Linux es considerado como un descendiente directo del So Unix.
Las caracteristicas principales del sistema operativo Linux son:
1.Es un sistema robusto.
2.Es estable.
3.Es multiusuario.
4.Es multitarea.
5.Es multiplataforma.
6.Tiene gran capacidad para gestion de redes.
7.Es libre.
